What is Javascript Prototype? All JavaScript object inherites proporties from Object prototype. From example: var Person=function(profession,name,age,gender){ this.profession = profession; this.name=name; this.age=age; this.gender=gender; this.alive=true; } // mike.prototype = new Person ("Mike",20); Here are some its most popular methods and their usage. mike.name will return 'Mike'; mike.gender will return undefined; mike.alive will return true; Now we will define a new object Developer. var Developer = function(language) { this.language = language; this.speak = function() { console.log("I love writing in " + this.language + "!"); }; } Developer.prototype = new Person("Developer"); var mike = new Developer("Ruby"); mike.speak(); will return "I love writing in Ruby!" mike.profession = "Developer"; Method: hasOwnProperty(), this method return a boolean value to see if the object have property pass to the parameter. mike.hasOwnProperty("language")#=> true mike.hasOwnProperty("alive")#=>false Even mike // also include comparison to ruby class inheritance with prototype properties in javascript
